# ProtegeLar
 
O ProtegeLar é um sistema desktop desenvolvido em C# com Windows Forms e SQLite para auxiliar empresas prestadoras de serviços residenciais no gerenciamento de clientes, serviços e orçamentos.
 
O sistema permite o cadastro e autenticação de usuários, recuperação de senha por e-mail, gerenciamento de serviços, criação de orçamentos, consulta ao histórico e geração de documentos em PDF.

## Funcionalidades
 
O ProtegeLar tem como objetivo facilitar o gerenciamento de serviços e orçamentos para empresas prestadoras de serviços residenciais. Para isso, o sistema possui funcionalidades como:
 
- Cadastro e autenticação de usuários
- Recuperação de senha por meio de envio de e-mail
- Gerenciamento de perfil do usuário
- Organização dos serviços por categorias
- Consulta automática de preços por metro quadrado
- Cálculo automático de valores para orçamentos
- Criação e gerenciamento de orçamentos
- Adição de múltiplos serviços em um mesmo orçamento
- Histórico de orçamentos realizados
- Alteração de preços dos serviços cadastrados
- Exclusão lógica de serviços (Soft Delete)
- Armazenamento de informações em banco de dados SQLite
- Geração de documentos PDF para apresentação dos orçamentos
- Controle e gerenciamento de clientes vinculados aos orçamentos
